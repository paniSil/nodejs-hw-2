# ДЗ #2 Node.js

Домашнє завдання до другої лекції з Node.js. Містить розв'язання до трьох задач, спрямованих на роботу з файлами.

## 1. Функція writeFileAsync

Функція для асинхронного запису данних в файл, та документування процесу запису та помилок.

## 2. Функція readFileAsync

Функція для асинхронного читання вмісту файлу. Підтримує логування у випадку успішного читання, чи у випадку помилок (в тому числі в разі, якщо файл не знайдено)

## 3. Функція deleteFileAsync

Функція для асинхронного видалення файлу. Підтримує логування у випадку успішного видалення, чи у випадку помилок чи відсутнього файлу.
